## Official Eve docs

Framework APIs, channels, connections, and the runtime live in the [Eve documentation](https://eve.dev/docs/introduction). Once `eve` is installed, the same docs ship at `node_modules/eve/docs`. Eve Directory guides cover install and adaptation only — they do not replace Eve’s channel and connection docs.
