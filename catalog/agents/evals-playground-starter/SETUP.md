# Set up Evals Playground Starter

## 1. Install

```bash
npm install
```

Provide model credentials the same way as any eve app (AI Gateway or provider key).

## 2. Run the suite

```bash
npm run eval
```

Read the eval files in order — each one is a self-contained lesson. Optional chat:

```bash
npm run dev
```

## Verify

Passing `npm run eval` is the whole point of this starter.
